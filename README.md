# CARLA_PythonAPI

Python-based client API for controlling and configuring the CARLA simulator.

This repository contains Python scripts for:
- Controlling ego vehicles in CARLA
- Integrated control of an ego vehicle with
  BehaviorAgent-based autonomous driving (auto control)
  and user-driven manual control via mode switching
- Configuring simulation environments (world, weather, traffic)
- Integrating input devices (keyboard, steering wheel)

## Environment

- CARLA 0.9.16
- Python 3.10.19
- OS: Windows 10 / 11
- GPU: NVIDIA GPU (tested on RTX 3060)

## Input Devices

- Keyboard
- Logitech Steering Wheel G920
